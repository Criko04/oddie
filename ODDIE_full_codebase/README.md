# ODDIE

**Operationally Disobedient Digital Intelligence Experiment**

Neutral chaotic AI personality powered by Raspberry Pi.

## Features
- Voice interaction
- Memory system
- GUI interface
- Expandable personality modules

## Wake/Sleep Commands
- Wake: `Hey Whiskerbyte`
- Sleep: `Good night Whiskerbyte`
